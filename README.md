## 📦 StockWise

StockWise is a lightweight inventory management app built for small business owners who
currently track stock manually. It replaces spreadsheets and notebooks with a simple,
mobile-first tool for managing products and monitoring stock levels in real time.

### Problem
Small shop owners often lose track of inventory — overselling out-of-stock items,
forgetting to reorder low-stock products, and wasting time on manual stock counts.

### Solution
StockWise gives business owners a fast way to add, view, update, and remove products,
with automatic low-stock alerts so nothing runs out unnoticed.

### Key Features
- **Add, edit, and delete products** with full form validation
- **Real-time inventory dashboard** — total products, total stock, low-stock count
- **Low stock alerts** for any item below 5 units
- **Search, filter by category, and sort** (name, quantity, newest)
- **Local persistence** — works fully offline via on-device storage

### Tech Stack
- **Framework:** Flutter (Material 3)
- **State Management:** Provider
- **Local Storage:** Hive
- **Platform:** Android

### Target Users
Small shop owners, market vendors, and micro-businesses who need a simple, no-backend-required
way to digitize inventory tracking.
